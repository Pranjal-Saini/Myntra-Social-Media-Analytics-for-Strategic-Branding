# Myntra Social Media Analytics 📊

An Excel-based analytics project focused on evaluating Myntra’s social media performance and generating actionable insights for marketing strategy.

---

## 🔍 Overview

Analyzed social media data across platforms like Instagram, YouTube, and Facebook to understand:

- Engagement trends  
- Content performance  
- Campaign effectiveness  
- Follower growth  

---

## 🛠 Tools Used

- Microsoft Excel  
- Pivot Tables  
- Charts (Line, Bar, Scatter)  
- Data Cleaning & Transformation  

---

## 📈 Key Analysis

- Calculated engagement rate across platforms  
- Identified top-performing posts & hashtags  
- Compared content types (videos, images, carousels)  
- Evaluated campaign performance  
- Analyzed follower growth vs ad spend  

---

## 💡 Key Insights

- Video content drives higher engagement  
- Campaigns boost reach and interaction  
- Hashtags improve visibility  
- Platform performance varies by content type
